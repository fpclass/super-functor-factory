isOutsideBounds :: Pos -> Bool Source #

isOutsideBounds pos determines whether pos is outside the boundaries of the conveyor belt and evaluates to True if that is the case or False otherwise.

makeEmpty :: Conveyor -> [Pos] -> Conveyor Source #

makeEmpty conveyor positions should turn all cells indicated by the given positions to empty cells.

advanceConveyor :: Conveyor -> Conveyor Source #

advanceConveyor conveyor should move all lambdas on conveyor as close to the end of the conveyor as possible.

nextMove :: Conveyor -> [Pos] Source #

nextMove conveyor should determine the next cluster on conveyor to process with the goal of maximising the Functor Factory's profit.
